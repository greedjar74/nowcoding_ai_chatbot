- pathCrossroad(): 아바타 앞, 뒤, 좌, 우측에 모두 길이 있는지 확인한다. 길이 있는 경우 True
- pathAheadRight(): 아바타 앞, 우측에 모두 길이 있는지 확인한다. 길이 있는 경우 True
- pathAheadLeft(): 아바타 앞, 좌측에 모두 길이 있는지 확인한다. 길이 있는 경우 True
- pathRightLeft(): 아바타 좌, 우측에 모두 길이 있는지 확인한다. 길이 있는 경우 True

---------------------------------------------------------------------------------------------------

- elif문: if문이 참이 아닐 때 다른 조건을 검사하는 데 사용한다.

[예시]
if 조건1:
    동작 코드
elif 조건2:
    동작 코드
else:
    동작 코드
