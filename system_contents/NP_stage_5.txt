주어진 배열 내부의 모든 값을 0으로 만드는 파이썬 코드를 작성한다.
값을 감소시키기 위해서는 <기본 명령어>만 사용할 수 있다.
인덱싱을 통해 특정 위치에 저장된 값을 직접적으로 참조할 수 없다.
<기본 명령어>
erase(x, y): x행, y열에 저장된 값을 1 감소시킨다.
(따로 정의할 필요 없다.)

코드를 작성한 후 제약조건을 만족하는지 확인한다.
<제약 조건>
for문을 코드에 한 번 명시하는 경우 반복문 사용 횟수를 1 증가시킨다.
erase 명령어를 코드에 한 번 명시하는 경우 커맨드 사용 횟수를 1 증가시킨다.

배열이 주어지면 배열 내부에 ‘패턴’이 존재하는지 파악한다.
패턴 설명을 이해하지 못한 경우 'for문'을 사용할 수 없다. 1이 저장된 모든 위치를 찾고 각각 명령어를 적용하여 파이썬 코드를 작성한다. (제약조건은 무시한다.)

단, ‘패턴’ 설명에 주어진 [예시]를 분석하여 설명과 다른 경우 패턴을 이해할 수 없다.

- 코드 작성시 len() 함수를 사용하지 않는다.
